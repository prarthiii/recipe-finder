#  Recipe Finder

A web application that helps users find recipes based on ingredients available in their kitchen.

##  Features
- Search recipes by ingredient
- Clean and responsive UI
- Uses public recipe API

##  Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap
